학생 등록 화면 리디자인 시안

동일한 '학생 등록' 화면을 서로 다른 두 디자인 철학으로 구현했습니다.
두 시안은 같은 데이터·같은 기능을 담되, 내비게이션 구조·색·정보 밀도에서 뚜렷이 갈립니다.

A형 · Teal Workbench

밀도형 업무 화면

한 화면에서 최대한 많은 정보를 빠르게 다루는 실무 집중형 레이아웃입니다.

  • 내비게이션상단 가로 탭 + 메가메뉴 드롭다운 (화면 폭을 세로로 더 확보)
  • 화면 구성좌측 목록 + 우측 상세 패널 동시 표시 · 가운데 핸들로 너비 조절
  • 색·톤민트(Teal) 포인트, 촘촘한 테이블, 컴팩트한 여백
  • 강점높은 정보 밀도 · 목록↔상세 즉시 전환 · 반복 입력에 유리
  • 추천 상황데이터 입·출력이 잦은 데스크톱 업무 담당자

B형 · Horizon

대시보드형 화면

여백과 카드로 정돈한 모던 대시보드형 레이아웃입니다.

  • 내비게이션좌측 접이식 사이드바 (대·중·소 메뉴를 한눈에 · 접어서 확장)
  • 화면 구성카드 단위 구획 + 우측 슬라이드 상세 패널 · 넉넉한 여백
  • 색·톤인디고(Indigo) 브랜드, 부드러운 그림자, 라운드 카드
  • 강점시각적 위계·가독성 · 메뉴 탐색 직관적 · 심미적 완성도
  • 추천 상황관리자·대시보드 뷰, 브랜드 인상을 중시하는 화면
